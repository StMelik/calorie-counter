# Счётчик калорий

## Описание
В этом проекте реализовано одностраничное приложение, которое работает в браузере — счётчик калорий. Приложение позволяет рассчитать индивидуальную суточную норму калорий.

## Функциональность
* Подсчет калорий.

## Технологии
* HTML
* CSS
* JavaScript
